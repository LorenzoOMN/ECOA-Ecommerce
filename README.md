## 📖 Sobre o Projeto

O **ECOA** é uma plataforma de e-commerce desenvolvida para uma marca de cerâmica artesanal que valoriza a sustentabilidade e o trabalho manual. O site permite que os clientes conheçam a história da marca, explorem o catálogo de peças únicas e entrem em contato diretamente via WhatsApp para finalizar a compra.

### ✨ Por que este projeto?

- **Sem transações online**: Para evitar fraudes e problemas de segurança, todas as vendas são finalizadas via WhatsApp
- **Design minimalista e pastel**: Paleta de cores que reflete a essência natural e sustentável da marca
- **Fácil gerenciamento**: Painel administrativo simples para cadastrar produtos e imagens
- **100% responsivo**: Funciona perfeitamente em desktops, tablets e smartphones

## 🚀 Funcionalidades

### Para Clientes
- ✅ Landing page institucional com história da marca
- ✅ Seção "Como Funciona" explicando o processo de compra
- ✅ Catálogo de produtos com fotos e descrições
- ✅ Integração direta com WhatsApp para cada peça
- ✅ Design responsivo e otimizado para mobile
- ✅ Cores em tons pastéis que transmitem sustentabilidade

### Para Administradores
- ✅ Painel administrativo protegido por senha
- ✅ Cadastro de produtos com upload de imagens
- ✅ Gerenciamento de preços e descrições
- ✅ Visualização em tempo real das alterações
- ✅ Armazenamento de imagens no Supabase Storage

## 🛠️ Tecnologias Utilizadas

### Frontend
- **[Next.js 14](https://nextjs.org/)** - Framework React com App Router
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Estilização com utility classes
- **[TypeScript](https://www.typescriptlang.org/)** - Tipagem estática
- **[Lucide React](https://lucide.dev/)** - Ícones modernos e minimalistas

### Backend & Banco de Dados
- **[Supabase](https://supabase.com/)** - Backend as a Service
  - PostgreSQL para banco de dados
  - Storage para imagens dos produtos
  - Real-time subscriptions

### Hospedagem
- **[Vercel](https://vercel.com/)** - Hospedagem gratuita e deploy automático
